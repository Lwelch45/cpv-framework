# Running tests

Here are the steps to run the unit tests, for now only ubuntu is supported.

``` text
git clone --recurse-submodules https://github.com/cpv-project/cpv-framework
cd cpv-framework
sh install-dependencies.sh
cd tests
sh run_tests.sh
```
